About ScriptGuesser – Q&A

What's ScriptGuesser?
ScriptGuesser is a guesser game like GeoGuessr, but instead of guessing Google Maps coverage, you guess writing scripts used around the world.
What's the purpose of this game?
Originally, this game was meant as a way for GeoGuessr players to learn and practice recognizing different writing systems and where they are are used.

However, this game can also be fun for anyone who is curious about writing systems or who wants to impress others with their ability to recognize languages.

Regardless of your motivation, please feel welcomed.
Why are some scripts missing from the config menu?
Since this game was meant for GeoGuessr players in the first place, only the scripts which can be found on GeoGuessr (on Google Maps) are present in the game right now.

I am planning to implement a system to select scripts from presets in the future though. Once I've done that, I will add more scripts.
How am I supposed to recognize the different scripts??
I will add a "Learning" page which shows example phrases and tips on how to spot the differences between the scripts in the future. In the meantime you can use Plonk It. You can find differences between the Indian scripts here for example.
Can I see the games source code?
Of course! The whole code of the game is open-source on GitHub. Since the project runs 100% on the client side, you can also inspect anything by opening the Developer Tools in your browser, which can in most browsers be done by pressing F12 on your keyboard.
How do I report issues with the game?
Please open a new Issue on GitHub and I will take a look at it as soon as I can.
